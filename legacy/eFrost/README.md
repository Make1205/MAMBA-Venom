# Legacy eFrost provenance code

This directory contains legacy ephemeral-variant code retained for provenance only.
It is not part of the current MAMBA-Frost implementation.
It is not used by the default build, Frost KAT generation, benchmarks, or paper data.
The current supported implementation is under `Frost/`.
The legacy files may contain upstream FrodoKEM/eFrodo naming and are not maintained as current Frost code.
